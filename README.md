# YOLO Annotation Tool (PyQt)

A **ready-made YOLO annotation application built with PyQt** designed to make dataset labeling faster, easier, and more efficient. This tool provides a simple graphical interface for drawing bounding boxes and generating YOLO-format annotations automatically.

---

## 🚀 Overview

Manually labeling datasets can be slow and repetitive. This tool streamlines the process by offering a clean PyQt GUI where you can:

- Load images
- Draw bounding boxes visually
- Assign class labels
- Automatically export YOLO annotations

The goal is to significantly reduce annotation time while maintaining accuracy and consistency.

---

## ✨ Features

- 🖥️ Built with **PyQt (GUI-based interface)**
- ⚡ Fast and responsive annotation workflow
- 🖱️ Click-and-drag bounding box creation
- 🏷️ Multi-class labeling support
- 📁 YOLO format export (`.txt` files)
- 💾 Save / auto-save annotations
- 📂 Easy dataset navigation (next/previous image)
- 🎯 Designed for YOLOv5 / YOLOv8 / YOLOv9 pipelines

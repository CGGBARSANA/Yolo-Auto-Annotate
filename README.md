# YOLO Annotation Tool

A **ready-made YOLO annotation application** designed to speed up dataset labeling and improve annotation efficiency. This tool simplifies the process of creating YOLO-format bounding box annotations, allowing you to focus more on model development and less on manual labeling work.

---

## 🚀 Features

- ⚡ Fast and efficient annotation workflow
- 🖱️ Simple click-and-drag bounding box creation
- 🏷️ YOLO format export support
- 📁 Easy dataset organization
- 💾 Auto-save and manual save options
- 🎯 Designed for speed and productivity
- 🧠 Ideal for object detection dataset preparation
